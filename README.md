# devenv

A glamorous shell scripts to install development tools, libraries,.. on arch, macosx, fedora and ubuntu

## 🚀 Features

* Git
* Rust
* Java
* Python
* Go
* Lua
* JavaScript
* Ruby
* Erlang
* Elixir
* Haskell
* Perl
* PostgreSQL
* SQLite
* NeoVim

## 📖 Usage

```shell
sh devenv.sh
```

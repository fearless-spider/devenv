# devenv

A glamorous shell scripts to install development tools, libraries,.. on arch, macosx, fedora and ubuntu

## Operating System

* Arch Linux
* Fedora Linux
* Ubuntu Linux
* MacOSX

## 🚀 Features

* Git
* Rust
* Java
* Python
* Go
* Lua
* JavaScript
* TypeScript
* Ruby
* Erlang
* Elixir
* Haskell
* Perl
* PostgreSQL
* SQLite
* MongoDB
* NeoVim
* LSP

## 📖 Usage

```shell
sh devenv.sh
```

## 📖 Usage

```shell
./devenv.sh              # interactive mode (default)
./devenv.sh --ci         # non-interactive CI smoke-test
./devenv.sh --uninstall  # remove previously installed components
```

Every run appends a session record to `~/.devenv_installed`.

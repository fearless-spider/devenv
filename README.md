# devenv

A glamorous shell scripts with [gum](https://github.com/charmbracelet/gum) to install development tools, libraries,.. on arch, macosx, fedora and ubuntu

## 🚀 Features

- Rust
- Java
- Python
- Go
- Lua
- JavaScript
- NeoVim

## 📖 Usage

```shell
sh devenv.sh
```
